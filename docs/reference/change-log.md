# Changelog

All notable changes to PhLynx will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added 

- Ability to populate workspace using Circulatory Autogen configuration files.
- Macrobuilder feature.
- Documentation and About windows.

### Changed

- Name changed from Circulatory Autogen Model Builder to Physiome Links (PhLynx).

## [0.1.0] - 2025-10-30

### Added

- Ability to import CellML module files
- Ability to place, edit, and connect module nodes
- Ability to export Circulatory Autogen configuration files.
