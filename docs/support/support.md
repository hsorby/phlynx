# Feedback & Support

We welcome contributions and feedback from the community to help improve PhLynx.

## Reporting Bugs
If you encounter unexpected behavior:
1.  Check the [FAQs & Troubleshooting](./faq-and-troubleshooting) guide.
2.  If the issue persists, please raise an issue on our [GitHub Repository](https://github.com/physiomelinks/phlynx/issues/new/choose).
    * *Please include steps to reproduce the error and any relevant screenshots.*

## Feature Requests
Have an idea for PhLynx? We'd love to hear it. 
Please submit a request via [GitHub Issues](https://github.com/physiomelinks/phlynx/issues/new/choose) and tag it with the label `enhancement`.

## Contributing
If you want to contribute code, please see our [Developer Guide](../guides/develop-phlynx).
